# Java and Android SDK - Quickstart

This quickstart provides step-by-step instructions on installing and setting up the Ditto SDK in your Android or Java project.
To integrate Ditto with your app:

1. Confirm that your project’s project-level build.gradle file includes the mavenCentral() repository. (Confirming Project-Level build.gradle)
2. Add the Ditto SDK module configurations and dependencies in the app-level build.gradle file. (Adding Ditto to App-level build.gradle)
3. Add the necessary permissions in your project’s AndroidManifest.xml file. (Configuring End-User Access Permissions)
4. Create a new instance of a Ditto class. (Importing Live Bindings and Instantiating Class Object)
